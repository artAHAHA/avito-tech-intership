# Автотесты для платформы модерации объявлений

Тестируется сайт: [https://cerulean-praline-8e5aa6.netlify.app/](https://cerulean-praline-8e5aa6.netlify.app/)

## Локальный запуск (Linux / macOS / WSL)


```bash
# Клонируем репозиторий
git clone https://github.com/artAHAHA/avito-tech-intership.git
cd moderation-tests

# Устанавливаем зависимости
pip install -r requirements.txt

# Запускаем тесты
python main.py
```


# Структура проекта
```
.
├── README.md                 # Инструкция по установке, запуску тестов и описание проекта
├── TESTCASES.md              # Список тест-кейсов с шагами и ожидаемыми результатами
├── BUGS.md                   # Отчёт о найденных багах 
├── requirements.txt          # Зависимости Python 
├── conftest.py               # Pytest-фикстуры
├── pages/                    # Папка с Page Object моделями
│   ├── base_page.py          # Базовый класс с общими методами (ожидания, клики, скроллы)
│   ├── main_page.py          # Методы для главной страницы (/list): фильтры, сортировка, цены
│   ├── stats_page.py         # Методы для страницы статистики (/stats): таймер, кнопки обновления
│   └── theme_switch.py       # Методы для переключения светлой/тёмной темы (мобильная версия)
└── tests/                    # Папка с тестами
    ├── test_main_page.py     # Тесты фильтров, сортировки, тогла "Только срочные"
    ├── test_stats_page.py    # Тесты управления таймером обновления статистики
    └── test_theme_switch.py  # Тест переключения темы на мобильной версии
```
# Устойчивость к изменению данных
Тесты не полагаются на фиксированное количество объявлений или конкретные ID. Все проверки основаны на актуальных данных со страницы (цены, категории, наличие флага «срочно»). Фильтры и сортировки проверяются математически (сравнение цен, проверка диапазонов), поэтому изменение базы данных другими пользователями не ломает тесты.
