# Тест-кейсы UI

## 1. Главная страница – фильтр «Диапазон цен»
**Предусловие:** Открыта главная страница `/` (или `/list`).

**Шаги:**
1. В поле «Цена от» ввести 5000.
2. В поле «Цена до» ввести 15000.
3. Дождаться автообновления.

**Ожидаемый результат:** В списке объявлений отображаются только товары с ценой от 5000 до 15000 включительно.

## 2.1. Главная страница – сортировка «По цене»
**Предусловие:** Открыта главная страница, есть хотя бы 2 объявления с разной ценой.
**Шаги:**
1. Выбрать в выпадающем списке сортировку «По цене», выбрать порядок «По убыванию».
2. Проверить порядок цен.

**Ожидаемый результат:** Объявления отсортированы по убыванию цены.

## 2.2. Главная страница – сортировка «По цене»
**Предусловие:** Открыта главная страница, есть хотя бы 2 объявления с разной ценой.
**Шаги:**
1. Выбрать в выпадающем списке сортировку «По цене», выбрать порядок «По возрастанию».
2. Проверить порядок цен.

**Ожидаемый результат:** Объявления отсортированы по возрастанию цены.

## 3. Главная страница – фильтр «Категория»
**Шаги:**
1. Выбрать категорию «Электроника» (или любую другую).
2. Нажать «Применить».
   
**Ожидаемый результат:** Показываются только объявления выбранной категории.

## 4. Главная страница – тогл «Только срочные»
**Шаги:**
1. Включить тогл «Только срочные».
   
**Ожидаемый результат:** Отображаются только объявления с пометкой «Срочно» (или с высоким приоритетом).

## 5. Страница статистики – таймер обновления
**Предусловие:** Открыта страница `/stats`.
**Шаги:**
- Нажать кнопку «Обновить» – статистика должна обновиться.
- Нажать кнопку «Стоп» – таймер перестаёт тикать, данные не меняются автоматически.
- Нажать кнопку «Старт» – таймер возобновляет обновление каждые N секунд.
  
**Ожидаемый результат:** Все кнопки работают согласно описанию.

## 6. Мобильная версия – переключение темы
**Предусловие:** Эмуляция мобильного устройства (ширина <= 768px).
**Шаги:**
1. Нажать на иконку/кнопку переключения темы (светлая/тёмная).

**Ожидаемый результат:** Цвет фона и текста меняются на противоположные. После перезагрузки страницы выбранная тема сохраняется.
