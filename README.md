# Тестовое задание на стажировку QA (весенняя волна)
В репозитории представлено мое решение тестового задания на стажировку QA.

# Структура проекта
```
.
├── Task1/
│   └── Task1.md               # Задание 1. Скриншот с багами
└── Task2/                     # Задание 2. Тесты UI 
    ├── README.md              # Инструкция по установке, запуску тестов и описание проекта
    ├── TESTCASES.md           # Список тест-кейсов с шагами и ожидаемыми результатами
    ├── BUGS.md                # Отчёт о найденных багах 
    ├── requirements.txt       # Зависимости Python 
    ├── conftest.py            # Pytest-фикстуры
    ├── pages/                 # Папка с Page Object моделями
    │   ├── base_page.py       # Базовый класс с общими методами (ожидания, клики, скроллы)
    │   ├── main_page.py       # Методы для главной страницы (/list): фильтры, сортировка, цены
    │   ├── stats_page.py      # Методы для страницы статистики (/stats): таймер, кнопки обновления
    │   └── theme_switch.py    # Методы для переключения светлой/тёмной темы (мобильная версия)
    └── tests/                 # Папка с тестами
        ├── test_main_page.py  # Тесты фильтров, сортировки, тогла "Только срочные"
        ├── test_stats_page.py # Тесты управления таймером обновления статистики
        └── test_theme_switch.py # Тест переключения темы на мобильной версии
```
